# Snapmaker Studio — 60–90s demo video script

> Independent open-source project — not affiliated with or endorsed by Snapmaker.
> Local-first; no cloud, no account. Redact any IP/hostname/path before publishing.

Goal: show a first-time U1 owner going from "I found a model" to "Studio prepared it and
can send it" — fast, plain-language, honest. ~75 seconds.

| Time | On screen | Voiceover (plain) |
|------|-----------|-------------------|
| 0:00–0:08 | Dashboard, hover the **Start your first print** card | "New to the Snapmaker U1? Studio walks you from a model to a print — no slicer knowledge needed." |
| 0:08–0:18 | **Find Models** → click an approved site; it opens in the locked Studio Model Browser | "Browse approved model sites inside Studio. Download an STL or 3MF." |
| 0:18–0:28 | **Source Check** on the file → detected slicer + can-read / needs-Orca / next step | "Studio tells you what the file is, what it can read, and the safe next step for your U1." |
| 0:28–0:40 | **Project Doctor** result → verdict + geometry findings | "It checks the real geometry — watertight, supports, bed fit — in plain language. Advisory, never a guarantee." |
| 0:40–0:50 | **Plate Color Remap** → pick plate + from/to colour → 2D preview | "Change one plate's colour and see exactly what changes and what stays protected. Your original is never touched." |
| 0:50–0:58 | **Print Quality Doctor** → symptom + Add file → evidence panel | "Bad print? Pick a symptom, add your file, and get advice grounded in your own model." |
| 0:58–1:05 | **Open in Snapmaker Orca** → the slice→export→send hint | "Prepare a clean U1 copy, then slice in Snapmaker Orca. Studio doesn't slice — Orca does." |
| 1:05–1:15 | **Printer Hub** monitoring (redact IP) → control card + confirm dialog | "Watch your U1 live, and send the sliced file — every action is confirmed. Studio never auto-starts a print." |
| 1:15–1:20 | Footer: local-only badge | "All local. No cloud, no account. Open source." |

Production notes:
- Capture at 1280×800+; hide/blur the printer IP and any window with personal content.
- Do NOT start a real print on camera unless supervised and approved.
- Keep claims advisory: no "100% success", no "Studio slices", no "autonomous control".
- If recording tooling isn't available, this script doubles as the shot list for stills.
